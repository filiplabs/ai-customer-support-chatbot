# AI Customer Support Chatbot

AI-powered customer support chatbot built with Node.js, Express, JavaScript, and the OpenAI API.

## Features

- AI-generated customer support responses
- Modern chat interface
- Node.js and Express backend
- Secure API key stored in environment variables
- Responsive frontend

## Technologies

- JavaScript
- Node.js
- Express
- HTML
- CSS
- OpenAI API
